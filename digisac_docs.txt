  ## Agendamentos
  Descrição: Os agendamentos são utilizados para programar a plataforma para fazer alguma ação em uma determinada data e horário. Com esta ferramenta, você poderá agendar uma abertura de chamado automática com algum cliente, uma notificação a um atendente de sua plataforma ou até fazer as duas ações simultaneame
    ### [GET] Listar agendamentos
      Endpoint: {{URL}}/api/v1/schedule?perPage=40
      Parâmetros de query:
        - perPage: 40

    ### [GET] Buscar agendamento
      Endpoint: {{URL}}/api/v1/schedule/{{scheduleId}}

    ### [PUT] Editar agendamento
      Endpoint: {{URL}}/api/v1/schedule/{{scheduleId}}
      Body:
        {
          "contactId": "{{contactId}}", //ID do contato
          "departmentId": "{{departmentId}}", //ID do departamento
          "files": [],
          "message": "{{text}}",
          "notes": "{{notes}}}}",
          "notificateUser": true, //true or false
          "openTicket": true, //true or false
          "scheduledAt": "{{2023-07-15 10:35:18}}",
          "userId": "{{userId}}"
        }

    ### [POST] Criar agendamento
      Endpoint: {{URL}}/api/v1/schedule
      Body:
        {
          "contactId": "{{contactId}}",
          "departmentId": "{{departmentId}}",
          "files": [{{arquivos}}],
          "message": "{{text}}",
          "notes": "{{note}}}}",
          "notificateUser": true, //true or false
          "openTicket": true, //true or false
          "scheduledAt": "{{2023-07-15 10:35:18}}",
          "userId": "{{userId}}"
        }

    ### [DELETE] Excluir agendamento
      Endpoint: {{URL}}/api/v1/schedule/{{scheduleId}}

  ## Chamados
  Descrição: Acontecerá todo o atendimento entre atendente e contato. Abaixo estão as imagens da estrutura do bate papo e todas as ferramentas disponíveis:
    ### [GET] Buscar chamados do contato
      Endpoint: {{URL}}/api/v1/tickets?where[contactId]={{contactId}}
      Parâmetros de query:
        - where[contactId]: {{contactId}}

    ### [GET] Buscar pelo contactid todos os chamados do cliente trazendo quem encerrou/transferiu
      Endpoint: {{URL}}/api/v1/tickets?query={"where":{"contactId":"{{contactId}}"},"attributes":["id","isOpen","protocol","origin","startedAt","endedAt"],"include":[{"model":"ticketTransfers","attributes":["action","fromDistribution"],"include":[{"model":"byUser","attributes":["id","name","email","status"]}]}],"perPage":50}

      Parâmetros de query:
        - query: {"where":{"contactId":"{{contactId}}"},"attributes":["id","isOpen","protocol","origin","startedAt","endedAt"],"include":[{"model":"ticketTransfers","attributes":["action","fromDistribution"],"include":[{"model":"byUser","attributes":["id","name","email","status"]}]}],"perPage":50}


    ### [POST] Transferência de chamado
      Endpoint: {{URL}}/api/v1/contacts/{{contactId}}/ticket/transfer
      Body:
        {
            "departmentId": "{{departmentId}}",
            "userId": "{{userId}}", //id do usuário SE for abrir direto para ele
            "comments": "{{comments}}" //comentario interno
        }

    ### [POST] Transferência de Transferência em massa
      Endpoint: {{URL}}/api/v1/contacts/ticket/bulk-transfer
      Body:
        {
            "departmentId": "{{departmentId}}", //id do departamento que vai receber os chamados
            "userId": "{{userId}}", //id do usuário que vai receber os chamados
            "comments": "{{comments}}", //comentário da transferência
            "ticketsSelectedId": [
                "{{idTicket}}", //id dos chamados
                "{{idTicket}}",
                "{{idTicket}}",
                "{{idTicket}}",
                "{{idTicket}}",
                "{{idTicket}}",
                "{{idTicket}}"
            ]
        }

    ### [POST] Abertura de chamado
      Endpoint: {{URL}}/api/v1/contacts/{{contactId}}/ticket/transfer
      Body:
        {
            "departmentId": "{{departmentId}}",
            "userId": "{{userId}}",
            "comments": "{{comments}}" //comentario interno
        }

    ### [POST] Fechar chamado
      Endpoint: {{URL}}/api/v1/contacts/{{contactId}}/ticket/close

    ### [POST] Fechar um chamado com assunto

  ## Contatos
  Descrição: Contatos são os números dos seus clientes que serão salvos dentro da Digisac.
    ### [GET] Listar todos os contatos
      Endpoint: {{URL}}/api/v1/contacts?perPage=40
      Parâmetros de query:
        - perPage: 40

    ### [GET] Listar contatos da conexão
      Endpoint: {{URL}}/api/v1/contacts?where[serviceId]={{serviceId}}?perPage=40
      Parâmetros de query:
        - where[serviceId]: {{serviceId}}?perPage=40

    ### [GET] Buscar contato
      Endpoint: {{URL}}/api/v1/contacts/{{contactId}}

    ### [GET] Buscar contato por número de uma conexão específica
      Endpoint: {{url}}/api/v1/contacts?where[data.number][$iLike]=%{{number}}%&where[serviceId]={{serviceId}}
      Parâmetros de query:
        - where[data.number][$iLike]: %{{number}}%
        - where[serviceId]: {{serviceId}}

    ### [GET] Buscar tags vinculadas a um contato
      Endpoint: {{URL}}/api/v1/contacts/{{contactId}}?include[0]=tags&include[1]
      Parâmetros de query:
        - include[0]: tags
        - include[1]: 

    ### [GET] Buscar contatos vinculados ao Campo Personalizado
      Endpoint: {{URL}}/api/v1/contacts?query={"include":[{"model":"customFieldValues","include":[{"model":"customField","where":{"name":"{{customFieldName}}"}}]}]}
      Parâmetros de query:
        - query: {"include":[{"model":"customFieldValues","include":[{"model":"customField","where":{"name":"{{customFieldName}}"}}]}]}

    ### [GET] Busca de contatos que possuem X campos personalizados preenchidos
      Endpoint: {{URL}}/api/v1/contacts?query={"include":[{"model":"customFieldValues","include":[{"model":"customField","where":{"$or":[{"name":"{{customFieldName1}}"},{"name":"{{customFieldName2}}"},{"name":"{{customFieldName3}}"},{"name":"{{customFieldName4}}"}]}}]}]}
      Parâmetros de query:
        - query: {"include":[{"model":"customFieldValues","include":[{"model":"customField","where":{"$or":[{"name":"{{customFieldName1}}"},{"name":"{{customFieldName2}}"},{"name":"{{customFieldName3}}"},{"name":"{{customFieldName4}}"}]}}]}]}

    ### [GET] Buscar contatos vinculados a Organização
      Endpoint: {{URL}}/api/v1/contacts?query={"include":[{"model":"person","required":"true","where":{"id":"{{organizationId}}"}}],"perPage":"30"}
      Parâmetros de query:
        - query: {"include":[{"model":"person","required":"true","where":{"id":"{{organizationId}}"}}],"perPage":"30"}

    ### [GET] Buscar contatos vinculados a Pessoa
      Endpoint: {{URL}}/api/v1/contacts?query={"include":[{"model":"person","required":"true","where":{"id":"{{personId}}"}}],"perPage":"30"}
      Parâmetros de query:
        - query: {"include":[{"model":"person","required":"true","where":{"id":"{{personId}}"}}],"perPage":"30"}
